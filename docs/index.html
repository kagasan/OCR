OpticalCircuitRecognition

手書き回路をカメラで撮影して認識したい．
使いかた
  1. 方眼紙上で回路図を書く．
  2. 赤枠で囲う．
  3. 画像中央に枠が収まるように撮影．
  4. 枠のサイズをフォームで設定．
  5. 画像を選択．
  6. 実行ボタンを押す．
現バージョンについて
  • 試験段階です．
    特に閾値の調整がよくありません．
  • 処理はブラウザ上で行っています．
    （画像のアップロードは行っていません．）
    そのため，マシンによって処理が遅くなる場合があります．
  • 配線のみ認識できるはず．
  • 最終的にはブラウザ上で手直しまでできるようにしたい．
  • 画像は640x480(余白付き)に変換されてから処理します．
設定フォーム
横に収まる正方形の数
縦に収まる正方形の数
