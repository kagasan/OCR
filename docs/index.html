OpticalCircuitRecognition

手書き回路をカメラで撮影して認識したい．
使いかた
  1. 方眼紙上で回路図を書く．
  2. 赤枠で囲う．
  3. 画像中央に枠が収まるように撮影．
  4. 枠のサイズをフォームで設定．
  5. 画像を選択．
  6. 実行ボタンを押す．
現バージョンについて
  • 試験段階です．
  • まず画像を640x480(余白付き)に変換する．
  • 次に赤色枠を検出する．
  • 配線があるであろう線の部分を走査する．
  • 濃い画素が多いなら回路図に反映する．
  • 最終的にはブラウザ上で手直しまでできるようにしたい．
設定フォーム
横に収まる正方形の数
縦に収まる正方形の数
